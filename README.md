# Musika SamaSama Filipino Festival 2026

Static website for the Musika SamaSama Filipino Festival.

## Pages
- `index.html` — main festival site
- `lineup.html` — Artist Lineup (Coming Soon)
- `dance-showdown.html` — Dance Showdown (Coming Soon)
- `food-market.html` — Food & Market (Coming Soon)

## How it's hosted
Plain static HTML served by GitHub Pages directly from the `main` branch (no build step).
To update: edit/replace the HTML files, commit, and push to `main`. The live site updates automatically.

Settings → Pages → Source: **Deploy from a branch** → Branch: **main** / **/ (root)**.
